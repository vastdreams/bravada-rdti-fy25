# Supporting Materials

This directory contains additional documentation that supports and strengthens the RDTI claim.

## Directory Structure

### `contracts/`
R&D-related contracts and agreements:
- Research agreements
- Development contracts
- Collaboration agreements
- Third-party R&D service contracts
- Licensing agreements related to R&D

### `patents/`
Intellectual property documentation:
- Patent applications
- Granted patents
- Patent search reports
- IP strategy documents
- Innovation disclosures

### `publications/`
Research publications and presentations:
- Research papers
- Conference presentations
- Technical articles
- White papers
- Case studies

## Document Naming Convention

Use format: `YYYY-MM-DD_[material-type]_[project-name]_[description].ext`

Examples:
- `2024-01-10_contract_research-agreement_university-partnership.pdf`
- `2024-03-15_patent-application_digital-twin-platform.pdf`
- `2024-06-20_publication_conference-paper_ai-routing.pdf`

## Linking to Projects

Each supporting material should reference:
- Related project(s) in `01-project-documentation/`
- Relevant evidence in `02-evidence/`
- Date range of relevance
