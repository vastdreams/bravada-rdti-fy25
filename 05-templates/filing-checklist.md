# RDTI Filing Preparation Checklist

## Phase 1: Project Documentation

### Project Identification
- [ ] All R&D projects identified and listed
- [ ] Project names and IDs assigned
- [ ] Project timelines established (start/end dates)
- [ ] Project scope clearly defined

### Project Descriptions
- [ ] Project description completed for each project
- [ ] Technical objectives clearly stated
- [ ] Technical uncertainty explicitly described
- [ ] Systematic investigation approach documented
- [ ] Technical advancement explained
- [ ] Outcomes and results documented

### Administrative Records
- [ ] Project charters or initiation documents collected
- [ ] Budget approvals documented
- [ ] Resource allocations recorded

---

## Phase 2: Evidence Collection

### Technical Evidence
- [ ] Code repositories and commits documented
- [ ] Design documents and architecture diagrams collected
- [ ] Test results and experimental data organized
- [ ] Technical reports and analysis prepared
- [ ] Prototypes and proof-of-concepts documented
- [ ] All technical evidence dated and properly named

### Financial Evidence
- [ ] Invoices for R&D expenses collected
- [ ] Equipment purchase receipts organized
- [ ] Software license invoices gathered
- [ ] Third-party contractor invoices collected
- [ ] Material and supply costs documented
- [ ] All costs linked to specific projects

### Time Records
- [ ] Timesheets for R&D activities collected
- [ ] Project time tracking reports prepared
- [ ] Labor cost allocations calculated
- [ ] Employee time logs organized
- [ ] Time records match project timelines

### Meeting Notes
- [ ] Project planning meetings documented
- [ ] Technical review meetings recorded
- [ ] Problem-solving sessions noted
- [ ] Progress updates documented

---

## Phase 3: Cost Calculation

### Cost Categories
- [ ] Direct labor costs calculated
- [ ] Material costs totaled
- [ ] Equipment costs identified
- [ ] Third-party R&D costs documented
- [ ] Overhead allocations calculated (if applicable)

### Cost Verification
- [ ] All costs linked to specific projects
- [ ] Cost periods match project timelines
- [ ] Financial evidence supports cost claims
- [ ] Time records support labor cost claims
- [ ] Cost totals verified and cross-checked

---

## Phase 4: Filing Preparation

### Forms Completion
- [ ] RDTI claim form completed
- [ ] All required fields filled
- [ ] Project descriptions attached
- [ ] Cost schedules prepared
- [ ] Supporting schedules completed

### Document Organization
- [ ] Cover letter prepared
- [ ] Document index created
- [ ] Evidence package organized
- [ ] All documents properly named
- [ ] Submission package assembled

### Review
- [ ] All information accurate and complete
- [ ] Evidence supports all claims
- [ ] Costs verified and correct
- [ ] Forms reviewed for errors
- [ ] Final review completed

---

## Phase 5: Submission

### Pre-Submission
- [ ] Submission deadline confirmed
- [ ] Submission method determined
- [ ] Required copies prepared
- [ ] Submission package finalized

### Submission
- [ ] Documents submitted to tax authority
- [ ] Submission receipt obtained
- [ ] Submission date recorded
- [ ] Confirmation received

### Post-Submission
- [ ] Submission receipt filed
- [ ] Submission date logged
- [ ] Follow-up reminders set
- [ ] Documents archived

---

## Phase 6: Correspondence Management

### Authority Communications
- [ ] All correspondence logged
- [ ] Questions from authority documented
- [ ] Responses prepared and submitted
- [ ] Assessment notices filed
- [ ] Final determination recorded

---

## General Checklist Items

- [ ] All documents dated (YYYY-MM-DD format)
- [ ] Descriptive filenames used
- [ ] Version control maintained
- [ ] Sensitive information protected
- [ ] Backup copies created
- [ ] Project structure maintained
