# Templates and Checklists

This directory contains reusable templates and checklists for RDTI filing preparation.

## Available Templates

### Project Documentation Templates
- **Project Description Template**: Standard format for documenting R&D projects
- **Technical Objectives Template**: Structure for describing technical goals
- **Uncertainty Statement Template**: Format for explaining technical uncertainty

### Evidence Templates
- **Evidence Index Template**: Standard format for organizing evidence
- **Time Record Template**: Format for documenting R&D time
- **Cost Summary Template**: Structure for cost documentation

### Filing Templates
- **RDTI Claim Form Template**: Base template for main filing form
- **Cover Letter Template**: Standard cover letter format
- **Correspondence Template**: Format for tax authority communications

## Checklists

### Filing Preparation Checklist
- [ ] All projects documented
- [ ] Evidence collected and organized
- [ ] Costs calculated and verified
- [ ] Forms completed
- [ ] Review completed
- [ ] Submission package prepared

### Evidence Collection Checklist
- [ ] Technical evidence collected
- [ ] Financial records organized
- [ ] Time records complete
- [ ] Meeting notes documented
- [ ] All evidence dated and named correctly

## Usage

1. Copy relevant template to appropriate directory
2. Fill in project-specific information
3. Follow checklist to ensure completeness
4. Review against requirements
5. Move completed document to appropriate directory

## Template Maintenance

- Keep templates up-to-date with current requirements
- Document any changes to templates
- Maintain version history
- Solicit feedback on template effectiveness
