# User Flow: Project Manager

> **Persona**: PM / Contracts Manager  
> **Primary Modules**: Jobs Hub, Variations, Day Sheets, BI

---

## Role Summary

Oversees active jobs from contract signing through completion.

| Responsibility | Module |
|----------------|--------|
| Manage job execution | Jobs Hub |
| Create/track variations | Variations |
| Review/approve day sheets | Day Sheets |
| Monitor profitability | BI |
| Client relationship | Files & Email |

---

## Daily Workflow

```
1. Open job dashboard
2. Check P&L card → margin on track?
3. Check Labour card → hours vs budget?
4. Check Variations card → pending approvals?
5. Check AI Insights → any warnings?
6. Review pending day sheets → approve/reject
7. Check emails → action items
```

### Key Dashboard Cards

| Card | What to Check |
|------|---------------|
| P&L Summary | Margin trending correctly? |
| Labour Status | Hours within budget? |
| Variations | Pending approvals > 14 days? |
| Day Sheets | Items awaiting review? |
| AI Insights | Red flags? |

---

## Key Workflows

### 1. Job Takeover (from Estimator)

```
Job Won → PM Assigned → Review Quote/Budget → Review Contract → Assign Supervisor → Kick-off
```

| Review Item | Why |
|-------------|-----|
| Quote breakdown | Understand pricing basis |
| Contract terms | Know obligations |
| Estimator notes | Context transfer |
| Risk areas | Prepare mitigation |

### 2. Variation Creation

```
Scope Change → Determine Type → Create Variation → Submit → Track Approval → Invoice
```

| Type | When | Evidence |
|------|------|----------|
| Scope Change | Fixed-price adjustment | Instruction + Quote |
| Do-and-Charge | T&M work | Signed day sheets |
| Mini-Project | New scope | Full quote |

### 3. Day Sheet Approval

```
Notification → Review Details → Check Signature → Approve/Reject → Allocate to Variation
```

| Check | Question |
|-------|----------|
| Workers | Correct people? |
| Hours | Reasonable? |
| Description | Clear? |
| Signature | Valid? |
| GPS | Near site? |

### 4. Progress Claim Review

```
End of Period → Accounts Creates Claim → PM Reviews → Approve → Submit to Client
```

| Review | Question |
|--------|----------|
| Work value | Matches progress? |
| Variations | All approved included? |
| Retention | Correct? |
| Evidence | Attached? |

## Success Metrics

| Metric | Target |
|--------|--------|
| Job margin | ≥ Quote margin |
| Variation approval | < 14 days avg |
| Day sheet turnaround | < 24 hours |
| Client disputes | < 5% of claims |
